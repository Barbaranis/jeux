Bienvenue dans le Jeu de Memory

Les Règles du Jeu :

  • Retournez deux cartes à la fois pour essayer de trouver une paire identique.
  • Chaque fois que vous trouvez une paire, vous gagnez des points!
  • Les cartes sont mélangées à chaque niveau.
  • Le jeu continue jusqu'à ce que le temps soit écoulé.
  • Si vous atteignez un score minimum par niveau, vous pouvez passer au niveau suivant!
  • Le meilleur score sera sauvegardé, alors essayez de battre votre record!
